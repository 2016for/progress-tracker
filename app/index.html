Progress Tracker

A progress tracker written in SASS/CSS with flexbox to be flexible and responsive out of the box.
This can be used to show the stages of a process to a user like a multi stage form or a timeline.

Usage

Follow the code example below for basic usage, each demo shows the first two steps as complete, the thrid step being active and the last two steps being inactive. Modifier classes are added to the ul for different styles.

  <ul class="progress-tracker">
    <li class="progress-step is-complete">
      <span class="progress-marker"></span>
    </li>
    <li class="progress-step is-complete">
      <span class="progress-marker"></span>
    </li>
    <li class="progress-step is-active">
      <span class="progress-marker"></span>
    </li>
    <li class="progress-step">
      <span class="progress-marker"></span>
    </li>
    <li class="progress-step">
      <span class="progress-marker"></span>
    </li>
  </ul>
